<h1>A CameraPi University project - S3</h1>

<p>Goal : Monitoring system with alam deactivation. The camera gives the possibility to activate or deactivate the alam with a digital pad. The GPIOs are a buzer, a digital pad (no LED). The camera detects the movements. If there is movement, the alarm is triggered. A prototype for the detection of faces exists.</p>

<pre>
Language used : Java, (JUnit, Maven, Tomcat 9, OpenCV, SQLite)
Software used : Eclipse

  Last update : 29-12-2016
</pre>

<p>For an idea of the project <a href="#preview">click here</a>. We were a team of five people in Computer Engineering. You can try the web interface at home, but it will miss the servo motor and the camera along with the Raspberry Pi 3.</p>

Enjoy :)

<h1>Projet d'ingénierie avec CameraPi - S3</h1>

<p>But : Système de surveillance avec désactivation de l'alarme. La caméra donne la possibilité d'activer ou de désactiver l'alarme avec un pad  numérique. Les GPIOs sont un buzer, un pad numérique (pas de DEL). La caméra détecte les mouvements. S'il y a un mouvement, l'alarme est déclenchée. Un prototype pour la détection des visages existe.</p>

<pre>
 Langage utilisé : Java, (JUnit, Maven, Tomcat 9, OpenCV, SQLite)
Logiciel utilisé : Eclipse

  Mise à jour le : 29-12-2016
</pre>

<p>Pour avoir une idée du projet <a href="#preview">cliquez ici</a>. Nous étions une équipe de cinq personnes en génie informatique. Vous pouvez essayer l'interface web chez vous, mais il manquera le servo-moteur et la camera allant avec le Raspberry Pi 3.</p>

Amusez-vous bien :)
